# Mini CRM

A Customer Relationship Management (CRM) Web Application built using Flask and SQLite.

## Features

- Add Customer
- Edit Customer
- Delete Customer
- Search Customer
- Dashboard Statistics
- Responsive Design
- Dark Slate and Orange Theme

## Technologies Used

- Python
- Flask
- SQLite
- HTML
- CSS
- JavaScript

d

## Run Project

pip install -r requirements.txt

python app.py

Open:

http://127.0.0.1:5000
